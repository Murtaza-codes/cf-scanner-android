## 🇬🇧 English

Android version inspired by the Python Cloudflare scanner, redesigned with a modern UI and multi-language support.

### ✨ Features
- 🌍 **Region-based scanning:** China (optimized), Global, Russia
- ⚡ **Live scan progress:** Circular + linear progress with real-time matrix logs
- 🧠 **Low-latency results:** Sorts and shows top clean IPs
- 📋 **Copy button:** Copy `IP:PORT` instantly from results
- 🏳️ **Country flags:** Proper flag icons (not emoji)
- 🌐 **Multi-language UI:** English, Persian, Russian, Chinese
- ↔️ **Direction aware:** Persian = RTL, others = LTR
- 📱 **Professional Android UI:** Jetpack Compose + modern components

### 📥 Download APK (Recommended)

Users should download the app directly from the **Releases** section:

- Open: [https://github.com/Murtaza-codes/cf-scanner-android/releases](https://github.com/Murtaza-codes/cf-scanner-android/releases)
- Download latest `*.apk`
- Install on Android device

> If Android blocks install, enable "Install unknown apps" for your browser/file manager.

### 🛠 Build from Source (Optional)

```bash
git clone https://github.com/Murtaza-codes/cf-scanner-android.git
cd cf-scanner-android
```

Open in Android Studio and run:

```bash
./gradlew assembleDebug
```

Release build:

```bash
./gradlew assembleRelease
```

APK output:

`app/build/outputs/apk/release/app-release.apk`
